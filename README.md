# My Expenses 💸

Fullstack expenses tracker app

## Tools 🛠

Nodejs v12.22.12
Ruby v3.0.3
Bundler v2.2.32

## How to install and run (development build) 🛠

```bash
docker-compose up --build # builds and starts rails and postgres containers
```
